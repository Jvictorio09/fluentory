How It Works

1. The sync fetches contacts from Infobip that were contacted in the specified time period.

2. It matches contacts to existing Leads by phone number.

3. Updates the last_contact_date field if the contact is newer.

4. Optionally creates new Leads for contacts not found in the CRM.

5. Automatically updates Lead status to "Contacted" if contacted within the last 7 days.

Command Line: You can also run the sync manually using: python manage.py sync_infobip_contacts --days=30 --create-new
